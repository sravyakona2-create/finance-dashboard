# Finance Dashboard UI

## Overview
This is a simple "Finance Dashboard" built using "HTML, CSS, and JavaScript".  
It allows users to track income, expenses, and overall balance, with interactive charts and role-based UI.


## Features
- Dashboard Summary: Balance, Income, Expenses
- Transactions Table: View, search, and filter transactions
- Role-Based UI: 
  - Viewer: View only  
  - Admin: Add/Delete transactions
- Charts: 
  - Pie chart for expense categories  
  - Line chart for balance trend
- Insights: Shows highest expense category


## How to Use
1. Open "index.html" in a web browser.
2. Switch roles using the dropdown.
3. Admin can add transactions using the form.
4. Search or filter transactions as needed.
5. View charts and insights updating automatically.


## Technologies Used
- HTML, CSS, JavaScript
- Chart.js for charts
- LocalStorage for data persistence
